## 如何封装 PO 模式
- 什么是 PO 模式
    页面对象模型，是把一个 HTML 页面封装成一个对象。通过调用对象的
    方法实现网页的交互。
    
- PO 模式的好处有哪些？？ Page Object
可读性，更加容易维护，分离了页面逻辑和测试逻辑， 让测试代码更加简洁，
页面操作可以重复调用。
- UI 自动化测试的行业标准
